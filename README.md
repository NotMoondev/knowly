## About

Random 3am made quiz game in Nuxt based on [OpenTriviaDB](https://opentdb.com/)

## Features

- Fetches questions from OpenTriviaDB API
- Supports multiple question types (True/False, Multiple Choice)
- Keeps track of user's score
- Displays questions and answers in a neat and organized manner

## Local development

1. Clone the repository
2. Install dependencies with `bun install`
3. Start the development server with `bun run dev`
4. Open a web browser and navigate to `http://localhost:3000`
